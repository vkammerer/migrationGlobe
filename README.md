See: [http://xseignard.github.io/migrationGlobe/](http://xseignard.github.io/migrationGlobe/)

To run it localy, run the following commands inside the root folder of the project: 

- `sudo npm install gulp -g`
- `npm install`
- `bower install`
- `gulp`
